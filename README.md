## Installation

### Install NodeJS
  - Download [NodeJS](http://nodejs.org)
  - Ensure you have administrator role when install to set the PATH environment variable
  
## Use Grunt
  - Open Command Line and run
    * npm install
    * npm install grunt-cli -g

### Development
- grunt
- public : http://localhost:3000/

### Release
- grunt release
- public : http://localhost:3000/

## Use Gulp
  - Open Command Line and run
    * npm install
    * npm install grunt-cli -g

### Development
- gulp
- public : http://localhost:3000/

### Release
- gulp release
- public : http://localhost:3000/
